# Чеклист перед публикацией в Google Play

## Обязательно

- [ ] Создать приложение в Play Console.
- [ ] Указать package name: `com.ltmfedory.roaminghelper`.
- [ ] Включить Play App Signing.
- [ ] Добавить подписанный AAB.
- [ ] Заполнить Data safety.
- [ ] Добавить privacy policy.
- [ ] Загрузить иконку 512×512.
- [ ] Добавить скриншоты телефона.
- [ ] Проверить приложение на Android 10, 12, 14, 15+.

## Data safety

Приложение не читает SMS и не отправляет данные на сервер.

Используемые разрешения:

- `ACCESS_NETWORK_STATE` — для проверки состояния подключения;
- `READ_PHONE_STATE` — только для отображения состояния сети/SIM, без чтения SMS и без передачи данных.

## Что важно написать в описании

Приложение помогает открыть настройки сети и роуминга. Оно не гарантирует регистрацию в конкретной сети и не обходит ограничения операторов.
