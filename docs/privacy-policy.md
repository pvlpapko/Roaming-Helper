# Privacy Policy

**Помощник роуминга** is a local Android helper app for opening mobile network, roaming and operator selection settings.

The app does not collect, store, sell or transmit personal data.

The app does not read SMS messages, contacts, photos, files, location or payment data.

The app may request `READ_PHONE_STATE` only to show basic local network information on the device, such as current network type, roaming status and operator name. This information is displayed only on the device and is not sent anywhere.

The app uses `ACCESS_NETWORK_STATE` to check local connection status.

No analytics SDK, advertising SDK or remote server connection is included.
